# Day 25｜2026-06-25｜run out 还是 run out of｜图片提示词

主题：run out 还是 run out of
知识点：`run out` 前面放用完的东西；`run out of` 前面放人，后面接用完的东西

统一风格：
小红书英语学习笔记风格，竖版 3:4，温暖米白纸张质感，黑色清晰大字，浅蓝和浅橙分别标记 run out / run out of，少量红叉、绿勾和荧光笔。每页只有一个视觉中心，留白充足，文字必须清楚可读。不要 PPT、杂志、商务汇报、深色科技、复杂人物、贴纸堆叠、密集表格、假手写英文。

## 图 1｜封面

主标题大字：
少了 `of`

中间两行对比：

底部小字：
`We ran out of milk.`

画面建议：
左边是打开冰箱发现空牛奶盒，右边是两个人从门口跑出去，形成误会反差。`of` 用红色圈出，浅蓝标记 run out，浅橙标记 run out of。

## 图 2｜场景对比

标题：
两个主语不一样

左侧：
`The milk ran out.`
牛奶用完了。
主语是“东西”。

右侧：
`We ran out of milk.`
我们没牛奶了。
主语是“人”。

画面建议：
左边突出 the milk，右边突出 we 和 of milk。

## 图 3｜一句话分清

标题：
一句话分清

核心内容：
`run out`
= 东西用完了
`Something runs out.`

`run out of`
= 人没了某样东西
`Someone runs out of something.`

画面建议：
两个便签并排，结构词加粗高亮。

## 图 5｜自测答案

标题：
自测答案

答案：
1. `ran out`
纸用完了。

2. `ran out of`
我们没纸了。

记忆口诀：
东西没了 → `run out`
人没东西了 → `run out of`

底部小字：
多一个 `of`，后面就要接“用完的东西”。
