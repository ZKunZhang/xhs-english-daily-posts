一人一句高考易错｜such后没名词就错

标题备选：

2. 一人一句高考易错｜so和such结构别背反
3. 一人一句高考易错｜such a和so别乱用

发布正文：
so 和 such 都能表示“如此”，但高考常考的是后面跟什么结构。

so + 形容词/副词：
so beautiful / so fast

such + 名词短语：
such a beautiful day / such an interesting book

对比：
She is so kind.
She is such a kind girl.

考场记法：so 看形容词/副词，such 看名词。

特别提醒：so many / so much / so few / so little 是常见特殊结构。遇到 many、much、few、little，别机械地只看后面有没有名词。
