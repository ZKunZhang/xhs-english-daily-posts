# 小红书英语单篇发布稿模板

适用于日更主线。默认产出一份 `完整发布稿.md`，包含选题判断、标题、正文、5 页图文文案、标签和自发评论。

## 数据依据

当前模板根据 2026-06-22 复盘调整：

- `多一个 the，人就进车里了`：272 观看、12.4% 点击率、30 收藏、6 涨粉。
- `wear / put on`：260 观看、12.6% 点击率、25 收藏、4 涨粉。
- `every day / everyday`：累计 2,617 观看，且仍在持续长尾增长。
- `look after / look for`：131 观看，冲突正确但后果不够强。
- `listen music`：累计约 35 观看，单点纠错不再作主线。

## 第一步：发布前去重

必须同时搜索：

1. `05_数据复盘/` 中最新的笔记明细表。
2. `04_内容归档/已发布/`。
3. `07_账号定位/笔记列表明细表_0525.md`。

只要知识点已发过，默认停止产出，不用换标题规避去重。

## 第二步：选题门槛

每项 1 分：

| 检查项 | 达标标准 |
| --- | --- |
| 高频 | 18—34 岁用户在口语、写作或生活中真会用 |
| 真对比 | A/B 两边都是成立的表达，不是正确项对错句 |
| 共同认知 | 中文意思接近，用户有合理的混淆动机 |
| 具体后果 | 可以用位置、动作、对象或时间变化呈现差异 |
| 一句话规则 | 不用语法术语也能讲清 |
| 可视化 | 能为两边各写一句短例句，画面明显不同 |

判断：

- `5—6 分`：进入日更主线。
- `4 分`：降为备选，必须先补强具体后果。
- `0—3 分`：暂停，不生成完整稿。
- 单点纠错、活动前缀、纯低频拼写差异：即使形式完整，也不进入主线。

## 第三步：标题路由

按优先级各生成 1 条：

1. `微小差异 + 具体后果`
   - 例：`多一个 the，人就进车里了`
   - 例：`off 换成 out，脱外套变成掏手机了`
2. `同一个中文意思 + 两种不同动作`
   - 例：`都是“穿”，一个说状态，一个说动作`
3. `A 和 B 原来不一样`
   - 仅在前两种写不出更强钩子时使用。

发布标题必须选最强具体后果，不自动选固定公式。标题只讲一个冲突，不叠加“高考易错”、“每日英语”或其他栏目词。

## 质量检查

- 英文拼写、介词、时态和例句自然度已核对。
- 标题给了具体后果，不只是说“不一样”。
- 封面在手机缩略图中仍能一眼看懂。
- 正文开头 2 句内已给出冲突。
- 固定 5 页，每页只完成一个任务。
- 自测题和答案不在同一页。
- 未发布重复知识点。
