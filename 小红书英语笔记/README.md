# 小红书英语笔记工作区

这个目录用于管理英语学习类小红书内容，从选题、制作、待发布、已发布到数据复盘统一放在这里。

## 当前入口

- `03_发布排期/当前排期.md`：当前唯一执行排期。
- `02_内容生产/待发布/主线/`：可直接发布或改稿的主线内容。
- `04_内容归档/已发布/`：已发布内容归档。
- `05_数据复盘/`：阶段性数据复盘和策略调整。
- `06_模板素材/`：提示词、模板和制图技能资料。
- `04_内容归档/活动项目/`：活动向内容和活动备选，不作为默认主线入口。
- `07_账号定位/分析结论_20260525.md`：账号定位和选题判断依据。

## 使用顺序

1. 先看 `03_发布排期/当前排期.md`。
2. 从 `02_内容生产/待发布/主线/` 找对应稿件；如果没有，就按 `06_模板素材/` 的模板补稿。
3. 发布后移动或记录到 `04_内容归档/已发布/`。
4. 每周把表现写入 `05_数据复盘/`，再调整当前排期。

## 清理规则

- 当前排期只保留一个入口，不再新增多个同级排期文件。
- 历史已发布稿只做去重参考，不直接回流到主排期。
- 活动资料只在明确参加活动时使用。
- 系统文件、空目录和临时文件不保留。
