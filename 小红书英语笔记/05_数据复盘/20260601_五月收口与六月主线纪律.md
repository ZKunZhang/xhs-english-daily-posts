## 六月前 3 天动作

### 1. 上午只放低门槛强对比

优先级：

1. `borrow / lend / keep`
2. `less / fewer`
3. `since / for`
4. `bring / take`
5. `look / see / watch`

这些比复杂结构更适合接住回落期流量。

### 2. 晚上内容降级为测试

晚上可以发：

- 评论选择题
- 旧爆款变体
- 高收藏结构题

但晚上内容低于 200 观看，不直接判定主线失败。

### 3. 不再让活动前缀抢标题

不要用：

```text
一人一句高考易错｜
高考必看｜
中考易错｜
```

优先用：

```text
A 和 B 原来不一样
A 不是 B
别再把 A 当成 B
```

## 复盘结论

本次结论：`微调`。

主线不变，调整的是选题顺序和标题纪律：

- 继续坚持 `A 和 B 原来不一样`。
- 6 月前 3 天优先低门槛熟词，不用复杂题测试账号上限。
- 高考活动只放晚上或内页，不抢上午标题。
- `be used to / get used to`、`have been to / have gone to` 继续证明结构辨析有效。
- 如果 06-01 至 06-03 上午主线都低于 `150` 观看，再进入 3 天小复盘，不提前换方向。

一句话结论：

5 月底是爆款回落，不是模型失效；6 月初要做的是守住 `原来不一样`，把更容易点开的熟词强对比放到上午主线。
