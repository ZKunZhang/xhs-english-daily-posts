some time 和 sometime 原来不一样

## 发布正文

很多人看到 `some time` 和 `sometime`，会以为只是空格写不写的问题。

但这两个在英文里，意思真的不一样。

先记住这个区别：

`some time` = 一段时间（两个词）

`sometime` = 某个时候 / 改天（一个词）

也就是说：

`some time` 重点在“时间长度”。

`sometime` 重点在“时间点”，但这个时间点不具体。

举个最常见的例子：

`I need some time to think.`

我需要一些时间想一想。

这里的 `some time` 是“一段时间”，强调需要多久。

`Let's meet sometime next week.`

我们下周找个时间见面吧。

这里的 `sometime` 是“某个时候”，强调时间还没定。

最容易错的是这两句：

不要写：

`I need sometime to finish it.`

应该写：

`I need some time to finish it.`

因为这里说的是“需要一段时间”，要用两个词 `some time`。

也不要写：

`Let's talk some time tomorrow.`

应该写：

`Let's talk sometime tomorrow.`

因为这里说的是“明天某个时候聊”，要用一个词 `sometime`。

一句话记住：

能换成 `a period of time`，就用 `some time`，两个词。

能换成 `at some point`，就用 `sometime`，一个词。

所以：

`Give me some time.`

给我一点时间。

`Call me sometime.`

改天给我打电话。

先收藏，下次写到 `some time` 和 `sometime`，先判断你想说的是“一段时间”，还是“某个时候”。
