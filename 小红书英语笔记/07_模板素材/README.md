# 模板素材说明

这里放生成内容和图片提示词时复用的模板、提示词和制图技能资料。

## 目录

- `prompts/`：生成小红书内容表或改写内容的提示词。
- `templates/`：发布稿、标题、图片提示词的结构模板。
- `skills/`：本项目内复用的制图/卡片技能资料。
- `说明文档/`：历史输出结构和强对比系列说明。
- `强对比辨析系列_内容表模板.md`：批量内容表模板。

## 使用规则

- 单篇内容优先用 `templates/`。
- 批量内容优先用 `prompts/xhs_contrast_series_prompt.md`。
- 历史说明只做参考，不作为新内容默认结构。
