## 发布正文

很多人看到 `any one` 和 `anyone`，会以为只是空格写不写的问题。

但这两个在英文里，重点真的不一样。

先记住这个区别：

`any one` = 任意一个（两个词）

`anyone` = 任何人 / 有人（一个词）

也就是说：

`any one` 重点在“一个”，通常是从一组人或东西里挑一个。

`anyone` 重点在“人”，意思是任何人、有没有人，不强调具体哪一个。

举个最常见的例子：

`You can choose any one of these photos.`

你可以从这些照片里任选一张。

这里的 `any one` 后面接 `of these photos`，是在一组照片里选一个。

`Anyone can join us.`

任何人都可以加入我们。

这里的 `anyone` 指任何人，不是在某一组里挑一个。

最容易写混的是这两句：

不要写：

`You can choose anyone of these photos.`

应该写：

`You can choose any one of these photos.`

因为 `photos` 是东西，不是人，而且这里是在“一组里选一个”。

再比如：

`Does any one know the answer?`

这句不如下面这句自然：

`Does anyone know the answer?`

因为这里想问的是“有人知道答案吗”，不是在强调某一组里的某一个。

一句话记住：

后面有 `of + 一组人/东西`，优先用 `any one`，两个词。

想说“任何人 / 有人”，用 `anyone`，一个词。

所以：

`Pick any one you like.`

选一个你喜欢的。

`Anyone can answer this question.`

任何人都可以回答这个问题。

先收藏，下次写到 `any one` 和 `anyone`，先判断你说的是“一组里某一个”，还是“任何人 / 有人”。
