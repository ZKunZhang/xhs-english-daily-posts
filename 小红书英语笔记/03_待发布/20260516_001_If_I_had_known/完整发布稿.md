# Day 06｜2026-05-16｜If I had known

主题：If I had known
知识点：过去虚拟：If + had done, would have done

## 发布标题

If I had known 后面怎么接？

## 标题备选

1. If I had known 后面怎么接？
2. 虚拟语气最常考的一个结构
3. 后悔过去，就用 would have done

## 发布正文

看到 `If I had known`，基本就在提示你：这是对过去的假设，主句常用 `would have done`。

图片内容如下，可按 5 张图发布：

第 1 页：
If I had known about the meeting, I ___ there.

A. will go  B. would have gone  C. will have gone  D. would go

第 2 页：
答案：B. would have gone

If I had known about the meeting, I would have gone there.

第 3 页：
过去事实相反的假设：

If + had done, 主句 + would have done

第 4 页：
例句：

If I had studied harder, I would have passed.

If she had called me, I would have helped her.

第 5 页：
这不是将来计划。

这是“当时没发生，现在后悔”。

自发评论：用 `If I had...` 写一句你的“早知道”。
