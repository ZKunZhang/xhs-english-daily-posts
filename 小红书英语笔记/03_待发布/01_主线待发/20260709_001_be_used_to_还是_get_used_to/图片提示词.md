主题：be used to 还是 get used to
知识点：be used to 和 get used to 的区别

## 图 2｜答案页

顶部写：
先看答案

中间放答案和核心内容：
be used to
= 已经习惯

get used to
= 逐渐习惯

所以这句要说：
I am still getting used to it.
