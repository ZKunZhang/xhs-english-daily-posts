主题：unless 还是 if not
知识点：unless 和 if not 的关系

统一风格：
小红书英语学习笔记风格，竖版 3:4，温暖米白纸张质感，浅黄高亮、浅蓝便签、少量红叉和绿勾，像博主手写整理的收藏型英语干货卡片。文字必须清楚可读，版面留白明确，不要商务风，不要 PPT 感，不要复杂人物，不要太硬的表格。

## 图 1｜封面页

主标题大字：
unless ≠ if not

副标题：
别再重复写 not

中间放题目：
You won't pass
unless you ___ hard.

## 图 2｜答案页

顶部写：
答案公布

中间放答案和核心内容：
unless = if ... not

所以 unless 后面
通常接肯定形式

You won't pass unless you work hard.

## 图 3｜规则页

标题：
一句话记住

核心内容：
看到 `unless`
先想成 `if ... not`

已经是否定
就别再重复写 `not`

## 图 4｜例句页

标题：
这样说更自然

例句内容：
I won't go unless he invites me.
除非他邀请我，否则我不去。

Unless it rains, we will go out.
除非下雨，否则我们会出去。

## 图 5｜避错页

标题：
别再写错啦

避错内容：
不要写成：
unless you don't work hard

正确表达：
unless you work hard

总结：unless 自带 not，后面别再补 not。
