## 发布标题

be used to 和 get used to 原来不一样

## 发布正文

很多人看到中文“习惯”，会把 `be used to` 和 `get used to` 混着用。

但英文里一个看结果，一个看过程：

`be used to` = 已经习惯
`get used to` = 逐渐习惯

从时间线的角度看，`be` 更像已经到位的状态，`get` 更像还在变化中的过程。

图片内容如下，可按 5 张图发布：

第 1 页：
I am still ___
the new schedule.

be used to
还是
get used to？

第 2 页：
先看答案：

be used to
= 已经习惯

get used to
= 逐渐习惯

所以这句要说：
I am still getting used to it.

第 3 页：
一句话记住：

be
看状态结果

get
看变化过程

第 4 页：
例句对比：

She is used to early mornings.

He got used to the cold weather.

第 5 页：
收藏复习：

如果你还在适应中
优先想
`get used to`

不是
`be used to`

后面继续整理这种“看着会，其实容易错”的英语点。
