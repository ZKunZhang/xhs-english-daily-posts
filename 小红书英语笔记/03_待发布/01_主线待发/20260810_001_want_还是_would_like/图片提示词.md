主题：want 还是 would like
知识点：want 和 would like 的区别

统一风格：
小红书英语学习笔记风格，竖版 3:4，温暖米白纸张质感，浅黄高亮、浅蓝便签、少量红叉和绿勾，像博主手写整理的收藏型英语干货卡片。文字必须清楚可读，版面留白明确，不要商务风，不要 PPT 感，不要复杂人物，不要太硬的表格。

## 图 1｜题目页

主标题大字：
want ≠ would like

副标题：
真的不是随便换

中间放题目：
___ a coffee, please.

你会选哪个？

## 图 2｜答案页

顶部写：
答案公布

中间放答案和核心内容：
want = 直接想要
would like = 礼貌想要

I would like a coffee, please.

## 图 3｜规则页

标题：
一句话记住

核心内容：
want 更直接；would like 更礼貌。

## 图 4｜例句页

标题：
这样说更自然

例句内容：
I want more time.
我想要更多时间。

I would like some tea.
我想要一些茶。

## 图 5｜避错页

标题：
别再写错啦

避错内容：
点餐或请求时，would like 更柔和。

错误示例：I want a coffee, please.
正确表达：I would like a coffee, please.

总结：want 直接；would like 礼貌
