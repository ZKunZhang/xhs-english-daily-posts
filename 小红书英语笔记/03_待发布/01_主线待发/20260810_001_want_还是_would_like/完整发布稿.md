want 和 would like 原来不一样

## 发布正文

很多人看到中文“想要”，会把 `want` 和 `would like` 混着用。

但英文里它们看的是两个不同角度：

`want` = 直接想要
`would like` = 礼貌想要

图片内容如下，可按 5 张图发布：

第 1 页：
want ≠ would like
真的不是随便换

___ a coffee, please.
你会选哪个？

第 2 页：
答案公布

want = 直接想要
would like = 礼貌想要

I would like a coffee, please.

第 3 页：
一句话记住

want 更直接；would like 更礼貌。

第 4 页：
这样说更自然

I want more time.
我想要更多时间。

I would like some tea.
我想要一些茶。

第 5 页：
别再写错啦

点餐或请求时，would like 更柔和。

错误示例：I want a coffee, please.
正确表达：I would like a coffee, please.

总结：want 直接；would like 礼貌

第 5 页底部小字：
后面继续整理这种“看着会，其实容易错”的英语点。
