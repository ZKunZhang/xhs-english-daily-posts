## 发布正文

很多人看到中文意思很像，就会把 `used to、be used to、get used to` 混着用。

但英文里它们不是随便互换，关键差别是：

used to ≠ be used to

这期重点记住：

过去常常 / 习惯于 / 变得习惯，三页对照

例子可以这样放：

- used to ≠ be used to
- 过去常常 / 习惯于 / 变得习惯，三页对照
- used to、be used to、get used to 的核心差别
