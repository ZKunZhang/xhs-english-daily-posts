# Day 72｜2026-08-20｜excited 和 exciting 原来不一样｜图片提示词

统一风格：封面角落小字建议：易混英语 39，小字处理，不抢主标题，后期在小红书系统里排。小红书英语学习笔记风格，竖版 3:4，像博主手写整理的英语干货卡片。背景是温暖的米白色纸张质感，搭配浅粉、浅黄、浅蓝便签元素，轻微马克笔高亮，排版有留白但不空。文字清楚可读，像手机截图收藏的学习笔记，不要商务风，不要PPT感，不要太硬的表格，不要复杂人物。每张图只讲一个重点，英文短语必须拼写准确。

图1题目页：主标题大字：excited ≠ exciting。副标题：真的不是随便换。中间放场景题：I am ___ about the trip. 底部小字：你会选哪个？整体像小红书英语博主的每日小测笔记，清爽、柔和、有收藏感，文字必须清晰可读。

图2答案页：顶部写：答案公布。中间放答案：excited = 人兴奋；exciting = 事物令人兴奋。下方放正确句子：I am excited about the trip.。把 excited、exciting 和核心中文解释用手绘荧光笔圈出来，旁边加贴纸感提示：别再混着用。

图3规则页：标题：一句话记住。核心内容：excited 形容人的感受；exciting 形容事情。 版面使用浅蓝便签和黄色高亮，像手写整理的英语笔记。

图4例句页：标题：这样说更自然。例句：She is excited.；The game is exciting.。每句下面放短中文解释：她很兴奋。；比赛很刺激。。把 excited、exciting 用浅黄色荧光笔高亮。

图5避错页：标题：别再写错啦。避错内容：人感到兴奋用 excited。 错误示例：I am exciting about the trip. 正确表达：I am excited about the trip. 底部总结便签：excited 说人；exciting 说事。使用红色手写叉和绿色手写对勾，柔和但重点明显。
