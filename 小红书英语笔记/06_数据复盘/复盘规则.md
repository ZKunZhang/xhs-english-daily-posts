# 复盘规则

目标：6 月先恢复并强化 `A 和 B 原来不一样` 账号模型，避免因为单日波动频繁改方向。

## 基本原则

- 每天记录，不每天决策。
- 3 天看趋势。
- 7 天改策略。
- 6 月前半月不轻易切主线，默认继续发 `A 和 B 原来不一样`。
- 评论少先不作为主问题，当前优先级是曝光、观看、涨粉。

## 每日轻复盘

频率：每天晚上或次日上午。

只记录，不做大调整。

记录字段：

```text
日期
当天总曝光
当天总观看
当天涨粉
上午主线标题
上午主线曝光
上午主线观看
上午主线点赞
上午主线收藏
上午主线涨粉
是否使用「A 和 B 原来不一样」标题
备注
```

每日判断只回答 3 个问题：

```text
今天是否守住「原来不一样」主线？
上午主线是否达到 200 观看？
当天涨粉是否接近 16？
```

每日不做这些动作：

- 不因为单日下滑就换栏目。
- 不因为评论少就改成评论测试。
- 不因为一条低于 200 观看就停掉主线。
- 不把高考/中考/活动包装重新放回主标题。

## 3 天小复盘

频率：每 3 天一次。

建议日期：

```text
06-03
06-06
06-09
06-12
06-15
06-18
06-21
06-24
06-27
06-30
```

判断维度：

- 3 天总曝光是否回升。
- 3 天总观看是否回升。
- 3 天净增粉是否接近日均 `16`。
- 是否至少有 1 条主线达到 `200+` 观看。
- 哪条主线收藏率最高。
- 上午发布是否明显好于晚上发布。

3 天小复盘可以调整：

- 后续 3 天的选题顺序。
- 封面第一句话。
- 是否把更低门槛熟词提前。

3 天小复盘不调整：

- 不切掉 `原来不一样` 主线。
- 不新增复杂栏目。
- 不把评论作为第一目标。

## 7 天正式复盘

频率：每周一次。

建议日期：

```text
06-07
06-14
06-21
06-28
06-30
```

正式复盘才允许改策略。

核心指标：

```text
周总曝光
周总观看
周净增粉
平均每日涨粉
观看最高的 3 条
收藏最高的 3 条
涨粉最高的 3 条
低于 200 观看的主线数量
```

目标线：

```text
单条普通达标：观看 200+
单条优质：观看 800+
单条爆款：观看 3000+
单日涨粉目标：16+
单周涨粉目标：112+
上午主线曝光恢复线：1500+
```

如果达标：

- 继续发 `A 和 B 原来不一样`。
- 把涨粉最多的题型复制到下一周。
- 保持上午主线发布时间。

如果不达标：

- 先换选题难度，不换账号主线。
- 优先改成更低门槛熟词强对比。
- 减少结构复杂、抽象语法、三词合集。
- 封面改得更直接，但标题仍保留 `原来不一样`。

## 异常处理

连续 3 天出现以下情况，才进入调整：

```text
上午主线曝光都低于 1000
上午主线观看都低于 150
单日涨粉连续低于 5
```

调整顺序：

1. 换更低门槛选题。
2. 缩短封面文字。
3. 上午只发最强主线。
4. 暂停晚上补量。
5. 7 天正式复盘后再考虑栏目变化。

## 投流复盘

1000 元一次性券只用于刚发且已经冒头的新主线。

投前门槛：

```text
发出 1-2 小时后，满足以下任意 2 条：
观看 100+
点赞+收藏 15+
收藏 5+
评论 1+
```

不投：

- 冷启动无互动的帖子。
- 晚上测试题。
- 单点纠错题。
- 偏活动包装的内容。

投后复盘：

```text
投后曝光
投后观看
投后涨粉
涨粉成本
收藏成本
是否带动主页其他笔记
```

## 结论口径

复盘结论统一用这 4 类：

```text
继续：数据达标，沿用当前主线。
微调：主线不变，只调选题顺序或封面。
降级：某类选题不适合作为上午主线，改为晚上补量。
暂停：连续低迷或偏离账号标签，暂时不发。
```
