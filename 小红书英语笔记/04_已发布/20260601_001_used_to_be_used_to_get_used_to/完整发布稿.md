## 封面钩子

used to ≠ be used to

## 发布正文

`I am used to get up early.`

这句很多人觉得没毛病，其实是错的。

`used to` 这个词，中文翻过来都是"习惯"，但用法分三种，混着用就翻车：

- used to do → 以前常做，现在不做了
- be used to doing → 已经习惯了
- get used to doing → 还在适应中

记一个偷懒办法：后面跟原形的，只能是 used to，讲的是过去；变成 be / get used to，后面就只能跟 doing。

所以开头那句要改成 `I am used to getting up early.`。

几个能直接搬去用的：

- 我以前每天一杯奶茶 → I used to drink milk tea every day.
- 现在习惯喝美式了 → I'm used to black coffee now.
- 还在适应早起 → I'm getting used to waking up early.

中间这两个最容易混：be used to 是已经习惯了的状态，get used to 是还在慢慢适应的过程。下次想说"我习惯了"之前先停一秒，分清是哪种。

后面继续整理这种"看着会，其实容易错"的英语点。
