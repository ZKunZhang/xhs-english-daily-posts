发布标题：
so 和 such 原来不一样

标题备选：
1. so 和 such 原来不一样
2. 一句话分清 so 和 such
3. 别再把 so 和 such 当一个词

发布正文：
这组不是意思难，而是结构最容易背反。中文里都像“这么、如此”，英文却要先看后面接的是形容词，还是名词短语。

`so` + 形容词/副词
`such` + 名词短语

一句话分清：`so` 看形容词副词；`such` 看名词。

对比：
She is so kind.
She is such a kind girl.

再注意：`so many / so much / so few / so little` 是常见特殊结构。遇到 `many、much、few、little`，别机械地只看后面有没有名词。

封面提示词：
