# 高考易错点活动更新规则

日期范围：2026-05-18 到 2026-06-22

## 更新边界

- 已经有 `图片/` 文件夹的内容视为已出图版本，不再更新 `内容.txt`。
- 没有 `图片/` 文件夹的内容可以轻量调整，优先只改标题、标题备选、封面大字和明显不顺的表达。
- 活动笔记看承接能力，不用曝光单独判断内容成败；重点看活动入口后的观看、互动、关注、评论和私信承接。

## 当前保留重点

- 强对比辨析优先保留：`A 和 B 原来不一样`
- 单点纠错里能重组为成对辨析的，优先重组，不再单独做弱标题
- 明显偏规则讲解、标题一眼就能猜到答案、且未发布的低质稿，已清理

## 未锁定内容调整方向

- 高频混淆词优先用 `A 和 B 原来不一样`。
- 固定短语优先用 `X 这个短语一定要记住`。
- 明确错误纠正优先用 `X 后面别再加/用 Y 了` 或 `别再写 X`。
- 活动识别不再强制放在标题最前面。优先让错误点或对比点出现在标题前 8-12 个字里。
- 需要保留活动感时，把 `高考易错` 放到封面小字、标签或正文首句，不要让它占掉标题最强点击位。
- 正文尽量补 1 句高考判题视角；能结合谓语形式、句子时间线或上下文语气顺手带出“为什么这个句子里更自然”就带出来，但不要硬写成术语课，也不要让人死记规则。

## 封面提示词优化方向

- 标题可以保留活动统一前缀，但封面不要重复写完整标题，只放一个最强冲突点。
- AI 只生成无字底图，不直接渲染标题；封面文字统一后期用小红书系统 `文字配图` 排。
- 封面按手机缩略图设计：所有关键信息集中在画面中间 70% 安全区，左下角和右下角留白，不放关键字。
- 后期排字最稳的是 3 组信息：顶部小字 `高考易错`，中间超大主标题，下方一行短副标题。
- 主标题优先用 `别写 X`、`X ≠ Y`、`X 后接 Y`，控制在 1-2 行。
- 减少 AI 味：不要复杂人物、贴纸、卡通插画、3D、渐变、PPT 模板、密集便签、满屏红绿标注。
- 视觉风格用真实纸质单页、正面俯拍、浅米白纸面、少量红笔/绿勾，高三错题感点到为止。
- 底图边角只放少量知识点相关的小物件，比如时钟、耳机、相机、衣架、日历、车票、书信、地图、答题卡，不要抢文字位置。
- 不要把例句、选项、评论引导放在封面底部；题目和解释放到第 2 页以后。

## 内页提示词优化方向

- 第 2-5 页统一走 `题目页 / 答案页 / 规则页 / 例句页 / 避错页`。
- 题目页只保留一道题，题干放中上部，底部留白，不要塞解释。
- 答案页先给答案，再给正确句；红叉标错句，绿勾标对句。
- 规则页只讲一句话，最多两行，不扩成整页语法课。
- 例句页只放 2 个短例句，配极短中文注释，像考前速记。
- 避错页只总结 1 个最容易错的点，底部可放 `考前再看` 或 `别再写错`。
