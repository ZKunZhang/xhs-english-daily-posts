# 角色定位｜2026-05-25 版本

这个目录用于替代“直接翻 `06_数据复盘`”的使用方式。

建议日常顺序：

1. 先看这份角色定位
2. 再看 Markdown 化的数据表
3. 原始 Excel 继续留在 `06_数据复盘`，只当数据源

## 当前账号角色

当前最适合的角色，不是泛泛的“英语学习号”，而是：

`高频英语易混点整理号`

更准确一点，可以写成：

`帮用户每天分清一个看着会、其实容易错的英语点`

## 当前已经跑通的主线

最稳的内容模型是：

- `XX 和 XX 原来不一样`
- 高频结构辨析
- 高频词 / 短语边界辨析
- 一句话讲清楚差别

当前最强验证点：

- `used to / be used to` 证明结构辨析能拉停留、收藏、涨粉
- `because / because of` 证明“结构 + 高频 + 一句话分清”能继续跑
- `in time / on time`、`job / work` 证明高频对比基础盘稳定

## 内容优先级

建议按这个顺序发：

- `60%`：结构辨析
- `30%`：高频双词 / 双短语辨析
- `10%`：短语收藏 / 固定搭配

## 标题和封面规则

- 主线标题继续用 `XX 原来不一样`
- 不把活动栏目型前缀放到标题最前面
- 封面优先直接放冲突点，不做重课程感封面
- 第 5 页继续固定一句：`后面继续整理这种“看着会，其实容易错”的英语点。`

## 活动策略

活动可以参加，但不要反过来让活动标题牵着主线走。

执行上：

- 主线发稿看 `03_待发布/01_主线待发`
- 活动稿优先看 `03_待发布/02_活动备选`
- 历史活动原稿看 `08_活动项目`

## 去重规则

去重优先级：

1. `06_数据复盘/笔记列表明细表-0525.xlsx`
2. `09_角色定位/笔记列表明细表_0525.md`

只要知识点已经发过，即使标题包装不同，也不再回到主线待发。
