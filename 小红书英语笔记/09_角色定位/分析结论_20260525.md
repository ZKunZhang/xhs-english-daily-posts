## 当前最重要的一句话

`used to 和 be used to` 仍然是第一拉盘内容，`because 和 because of` 是新跑出来的小苗，`travel 和 trip` 暂时还不能判断。

## 主线已经验证通过的内容

### 第一主线

结构辨析比普通词汇辨析更容易带来：

- 停留
- 收藏
- 关注

优先级最高的模型是：

- `used to / be used to`
- `stop doing / stop to do`
- `have been to / have gone to`
- `be made of / be made from`
- `because / because of`

### 第二主线

普通高频词 / 短语辨析继续保留，但不再是最优先：

- `job / work`
- `in time / on time`
- `travel / trip`
- `bring / take`
- `rise / raise`

### 第三补充线

短语收藏线不一定条条爆，但对涨粉和长期价值有用：

- `by mistake`
- `in advance`
- `as a result`
- `from now on`
