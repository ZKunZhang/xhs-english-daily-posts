## 当前最重要的一句话

`used to 和 be used to` 仍然是第一拉盘内容，`travel 和 trip` 已经证明高频词义辨析还能继续拿规模，`because 和 because of` 是最值得继续放大的结构新苗。

## 主线已经验证通过的内容

### 第一主线

高频结构辨析最容易带来：

- 停留
- 收藏
- 关注

优先级最高的模型是：

- `used to / be used to`
- `stop doing / stop to do`
- `have been to / have gone to`
- `be made of / be made from`
- `because / because of`

### 第二主线

普通高频词 / 短语辨析不是次要补充，而是现在仍然稳定拿规模的一条并行主线：

- `job / work`
- `in time / on time`
- `travel / trip`
- `bring / take`
- `rise / raise`

### 第三补充线

短语收藏线不一定条条爆，但对涨粉和长期价值有用：

- `by mistake`
- `in advance`
- `as a result`
- `from now on`

## 标题策略

最稳的标题框架还是：

`XX 原来不一样`

执行上注意：

- 不长期把活动前缀放在标题最前面
- 不做太课程感、太说明书感的封面
- 封面优先直接展示冲突点或填空场景
- `错句纠错` 可以做，但更适合作为补充，不适合作为长期主标题模型

## 内容比例

后续待发布按这个比例配：

- `45%` 高频词 / 易混辨析
- `35%` 结构辨析
