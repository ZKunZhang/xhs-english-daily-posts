# Day 22｜2026-06-01｜be used to 还是 used to

主题：be used to / used to
知识点：过去常常和现在习惯不是一个结构

## 发布标题

一句话分清 used to 和 be used to

## 标题备选

1. 一句话分清 used to 和 be used to
2. used to 和 be used to 很多人一直用错
3. be used to 和 used to 原来不一样

## 发布正文

这组不是不会背，而是一写句子就容易混。因为它们差的不是一个小语法点，而是整个时间感。

图片内容如下，可按 5 张图发布：

第 1 页：
她以前住这里
和
她现在已经习惯住这里

英文会一样吗？

第 2 页：
先记这两句：

She used to live here.

She is used to living here.

第 3 页：
一句话分清：

`used to do`
= 过去常常做

`be used to doing`
= 现在习惯于做

第 4 页：
例句对比：

We used to eat out.

We are used to eating at home.

第 5 页：
收藏复习：

先看有没有 `be`
再看是“以前常常”
还是“现在习惯”

看到 `used to`
不要急着往后写
先判断整句话在说哪个时间状态
