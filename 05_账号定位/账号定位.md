# 账号定位

更新日期：2026-06-22

## 核心定位

`生活场景英语易混点整理号`

每天帮用户分清一组“看着会，其实容易用错”的高频英语表达。

内容不先讲抽象规则，而是先呈现用户当天就可能遇到的生活动作或错误后果，再用一句话讲清区别。

## 已验证的内容模型

### 第一主线：具体后果型强对比

把很小的语言差异翻译成明确的生活后果。

已验证样板：

- `多一个 the，人就进车里了`
- `wear / put on`：穿着的状态 / 穿上的动作
- `every day / everyday`：使用位置不同，长尾表现突出

选题必须尽量满足：

1. 两边都是用户见过的高频表达。
2. 中文里容易被混成一个意思。
3. 英文中确实存在使用边界。
4. 能用两个可视化场景直接展示区别。
5. 能用一句话讲清，不需要先理解术语。

### 第二主线：高频结构辨析

用于拉收藏和关注，但封面仍要先呈现场景或结果。

例如：

- `used to / be used to`
- `because / because of`
- `in time / on time`
- `stop doing / stop to do`

### 补充内容

- 每 5—10 条发 1 条合集或评论区答疑。
- 普通固定短语只有在能改成强冲突时才进入主线。
- 单点纠错和活动包装不作为日更主线。

## 内容比例

- `70%`：低门槛、可视化的熟词强对比。
- `20%`：高频结构辨析。
- `10%`：合集、答疑或实验内容。

## 标题与封面

标题优先级：

1. `微小差异 + 具体后果`，例如“多一个 the，人就进车里了”。
2. `A 和 B 原来不一样`，用于冲突明确但暂时没有更强场景的选题。
3. 错句纠正，只作补充。

封面规则：

- 优先呈现两个生活动作或位置变化。
- 主标题和短副标题即可，不放密集规则。
- 不用“高考易错”、“每日英语”等栏目词抢占主标题。
- 第 2—5 页再完成例句对比、一句话规则和自测。

## 复盘与调整

- 每天记录，不因一天波动换主线。
- 3 天调选题顺序和封面。
- 7 天才调整内容比例和策略。
- 优先看曝光、观看、点击效率、收藏和涨粉；评论数不是当前第一目标。
