# Day 23｜2026-06-02｜too 还是 either

主题：too / either
知识点：肯定句和否定句的“也”不一样

## 发布标题

too 和 either 原来不一样

## 标题备选

1. too 和 either 原来不一样
2. 一句话分清 too 和 either
3. 别再把 too 和 either 当一个词

## 发布正文

中文里都能说“也”，但英文不会只给你一个词。最容易错的地方就是：句子一否定，词也要跟着变。

图片内容如下，可按 5 张图发布：

第 1 页：
这两句最后一个词
会一样吗？

I like it, too.

I don't like it, too / either

第 2 页：
先看答案：

`too` = 肯定句末的“也”
`either` = 否定句末的“也”

所以要说：
I don't like it, either.

第 3 页：
一句话分清：

肯定用 `too`
否定用 `either`

第 4 页：
例句对比：

She can sing, too.

She can't sing, either.

第 5 页：
收藏复习：

看到 `don't / doesn't / didn't / can't`
句末优先想 `either`

再补一句：
`also` 常放句中
`too / either` 常放句末

自发评论：你以前会把 `too` 和 `either` 混着用吗？
